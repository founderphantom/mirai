## Animation Principles

### Micro-interactions
- Button hover: Scale 1.02, duration 150ms
- Focus ring: Fade in 100ms, width 2px
- Loading spinner: Rotate 360deg, duration 1s, infinite

### Page Transitions
- Fade: Opacity 0 to 1, duration 300ms
- Slide: TranslateX -20px to 0, duration 500ms
- Scale: Scale 0.95 to 1, duration 300ms

### Stagger Effects
- List items: 50ms delay between elements
- Card grids: 75ms delay between cards
- Menu items: 30ms delay between items
